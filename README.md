## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- Code Editor of your choice.
- A browser
- Node.js environment

### Setup
- To clone this repo, run the following command: `git clone https://github.com/WilhelmK109/My-Portfolio.git`
### Install
- Run `npm install` to download all the project dependencies.
